# 菅野小児科医院ホームページ

このフォルダには、菅野小児科医院の公式ホームページ一式が含まれています。

## ファイル構成

- `index.html`: ホームページ本文
- `styles.css`: デザイン・レスポンシブ表示
- `script.js`: メニュー、診療時間案内、画像未設定時の表示処理
- `clinic-exterior.jpg`: 医院外観写真用のファイル名（現在は未配置）

## GPTへの依頼例

「ZIP内の菅野小児科医院ホームページを確認し、デザイン・文章・スマホ対応についてレビューしてください。」

外観写真を表示する場合は、同じフォルダへ `clinic-exterior.jpg` を配置してください。
